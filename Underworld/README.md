# Underworld

A cross-platform Avalonia UI application for managing DooM executables, IWADs, and game configurations.

## Build & Run

```bash
# Build
dotnet build

# Run
dotnet run --project Underworld

# Test
dotnet test Underworld.ViewModelTests
```

## Dependencies

- **Avalonia** 11.3.9: Cross-platform UI framework
- **CommunityToolkit.Mvvm** 8.2.1: MVVM utilities (ObservableObject, SetProperty)
- **xUnit** 2.5.3: Unit testing framework
